- Var thường được dùng ở những code cũ, hoặc những đoạn code thiếu chặt cần tính linh hoạt

- Let được sử dụng nhiều nhất vì nó không quá lỏng lẻo nhưng cũng có tính linh hoạt

- Const thường được những lập trình viên có kinh nghiệm sử dụng vì nó có thể cố định được biến không thay đổi
